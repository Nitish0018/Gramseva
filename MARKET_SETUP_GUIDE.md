## Quick Start

### 1. Access the Real-Time Dashboard
1. Open the application in your browser
2. Navigate to the "Market" tab in the bottom navigation
3. In the header, toggle from "Standard" to "Real-time" view
4. You'll see the enhanced dashboard with live updating prices

### 3. Setting Up Notifications

#### Enable Browser Notifications
1. When prompted, click "Allow" for desktop notifications
2. Or manually enable in browser settings for the site

#### Notification Types
- **Price Spikes**: Automatic alerts for changes >5%
- **Volume Alerts**: High trading volume notifications
- **Connection Status**: Online/offline indicators

#### Viewing Alerts
1. Click the bell icon in the top-right corner
2. Red badge shows unread alert count
3. Click to view detailed alert panel

## Troubleshooting

### Common Issues

#### 1. No Data Showing
- Check internet connection
- Refresh the browser
- Try switching to Standard view and back

#### 2. Notifications Not Working
- Enable browser notifications in settings
- Check if notifications are blocked
- Test with the notification button

#### 3. Slow Updates
- Check network connectivity
- Clear browser cache
- Close other bandwidth-intensive applications

#### 4. Mobile Display Issues
- Rotate device to landscape mode
- Zoom out if cards appear cut off
- Use two-finger pinch to adjust

### Performance Tips

#### For Best Experience
1. **Stable Internet**: Ensure good connectivity for real-time updates
2. **Modern Browser**: Use latest Chrome, Firefox, Safari, or Edge
3. **Enable JavaScript**: Required for real-time functionality
4. **Allow Notifications**: For important market alerts
